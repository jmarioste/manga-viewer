Features

  • Cross platform application for OSX / Linux / Windows
  • Supports .zip,.rar, .cbr file formats
  • Material Design UI
  • Bookmark a Folder for easy navigation
  • Add your favorite manga to favorites
  • Quickly search a folder and its subfolders for manga
  • Scans a directory and automatically lists the available manga from that directory.
  • Drag and Drop files to UI
